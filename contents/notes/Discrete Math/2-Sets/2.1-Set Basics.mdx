## Sets
Are an unordered non-repeating collection of objects or *elements*.

## Cardinality of Sets
The total number of elements within a set.

## Subset
The set that contains some or all elements of a given set.

## Proper Subset
The set that contains some and not all elements of a given set.

## Empty Set
The set that contains no elements.

## Set Equivalency
Two sets are said to be the same if they contain the exact same 
elements.

## Universal Set
The set in which other sets are considered into.

## Power Set
The set that contains all possible subset of a given set.
