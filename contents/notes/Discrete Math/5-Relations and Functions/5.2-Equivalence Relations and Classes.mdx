## Equivalence Relations
A relation that is *reflexive*, *symmetric*, and *transitive* 
all at once.

## Equivalence Classes
Is a set where for an equivalence relation *R* over some set *S*,
then for some element *a* in *S* then the equivalence class of *a* 
is:

$$[a] = \{x \in S | a R x\}$$
