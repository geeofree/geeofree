## Graphs
Are a set of vertices and edges.

## Adjacent Vertices
Are when two vertices that are joined together, making an edge.

## Neighbors
The set of adjacent vertices for some vertex.

## Incident
The edge that joins two vertices.

## Adjacent Edges
Are edges that have a common vertex.

## Graph Order
The total number of vertices in a graph.

## Graph Size
The total number of edges in a graph.

## Trivial Graph
A graph that contains a single vertex only.

## Empty Graph
A graph that contains no edges.
