## Operating System (OS)
A complex piece of software that manages the resources of 
a computer system and provides a convenient interface for 
users to interact with a computer system.

## Kernel
The main program in the OS that interacts directly with the 
computer resources and is the middleman between application 
programs and the computer resources that the application 
program uses.
